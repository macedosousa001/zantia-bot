# Zantia Bot — Webhook Telegram

Bot de apoio técnico da Zantia Formação com IA.

## Variáveis de ambiente necessárias no Railway:
- `TELEGRAM_TOKEN` — token do @ApoioZantiaBot
- `ANTHROPIC_API_KEY` — chave da API do Claude

## Setup:
1. Deploy no Railway
2. Definir as variáveis de ambiente
3. Registar o webhook no Telegram
